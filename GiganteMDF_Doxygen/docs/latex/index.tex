O projeto Gigantes de MDF, vem com o intuito dos alunos aplicarem os conceitos recebidos pela disciplina de Programação de Hardware. O intuito é cada grupo criar seu código de um robô que dispara laser e também detecta laser recebido por meio de sensores. Os componentes utilizados no projeto foram\+:


\begin{DoxyItemize}
\item 1x Arduino Nano
\item 1x Resistores de 10kohms
\item 3x Resisrores de 360ohms
\item 5x Botões
\item 2x Motores DC
\item 1x C.\+I de ponte H L293D
\item 1x Diodo Laser
\item 1x LDR de 20mm
\end{DoxyItemize}

O código foi desenvolvido na linguagem C utilizando apenas o ATMEGA328, foi utilizado o programa Simul\+Ide para debugar o codigo, também o próprio programa Arduino para gravar no arduino e testar.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Pinagem}{Pinagem}}\label{index_autotoc_md3}
